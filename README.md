# ChartLore

ChartLore is a trading journal web app built with Next.js for logging trades, reviewing performance, and analyzing execution patterns across dashboard, reports, strategy, analytics, and settings workflows.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` starts the local app
- `npm run build` creates a production build
- `npm run lint` runs ESLint
